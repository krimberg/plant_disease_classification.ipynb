# Классификация состояния растений по мультиспектральным снимкам с дронов

**Бинарная и многоклассовая классификация** здоровых/больных растений, стресса от засухи, переудобрения и других состояний.

### Цель и мотивация
Разработка модели глубокого обучения для автоматического мониторинга здоровья сельскохозяйственных культур с помощью снимков с дронов.  
Это позволяет своевременно выявлять проблемы, снижать потери урожая и оптимизировать использование удобрений и воды.

### Использованные данные
- **Основной датасет для разработки**: [PlantVillage Dataset](https://www.kaggle.com/datasets/abdallahalidev/plantvillage-dataset) — 54 306 изображений листьев, 38 классов (14 культур, healthy + 37 болезней).
- **Целевые данные**: Мультиспектральные и гиперспектральные снимки с дронов — [Beyond Visible Spectrum: AI for Agriculture 2025](https://www.kaggle.com/competitions/beyond-visible-spectrum-ai-for-agriculture-2025) (hyperspectral imagery для диагностики болезней).

### Архитектура модели и обоснование выбора
- **Основная модель**: ResNet50 (предобученная на ImageNet)  
  Обоснование: Хорошо работает на изображениях растений, устойчива к переобучению, легко адаптируется под многоканальные данные (изменение первого свёрточного слоя под 5+ каналов).
- Альтернативы, которые можно протестировать: EfficientNet-B0, Vision Transformer (ViT).

Для мультиспектральных данных добавляются вегетационные индексы (NDVI, NDRE и др.) как дополнительные каналы.

### Метрики качества
- Accuracy
- F1-score (weighted)
- Confusion Matrix
